# Multi-City Report Automation (Python)

## Project Description
This Python automation script reads data from a CSV file and generates separate city-based reports automatically. It also creates a summary report and maintains execution logs.

This project demonstrates real-world automation skills used in offices, freelancing, and backend data processing.

---

## Features
-Read CSV data
-Filter records by multiple cities
-Generate one report per city
-Create a summary report
-Maintain logs with timestamps
-Automatic folder creation

---

## Technologies Used
-Python 3
-CSV module
-File handling
-Datetime module
-OS module

---
